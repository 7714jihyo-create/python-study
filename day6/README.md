## 📦 파이썬 모듈 (Module)

### 1. 모듈의 정의
*   함수, 클래스, 변수 등을 정의한 **파이썬 소스 코드 파일**
*   **호환성:** 다른 파이썬 파일에서 `import`를 통해 작성된 함수나 클래스를 쉽게 가져다 쓸 수 있음.
*   **확장성:** 파이썬 표준 라이브러리나 외부 개발자가 만든 라이브러리를 활용할 때 자주 사용
    *   *예: `csv.py`라는 파일이 있다면 이를 **csv 모듈**이라고 부름.*

---

### 2. 모듈을 사용하는 이유
1.  **코드의 재사용성:** 동일한 코드 반복해서 작성하지 않고, 이미 만들어진 소스코드 다시 불러와 사용.
2.  **구조적 관리:** 비슷한 역할을 하는 클래스나 함수를 하나의 파일로 모아 관리 -> 유지보수 편리.
3.  **네임스페이스 제공:** 코드 간의 이름 충돌을 방지하고 기능을 체계적으로 분류.

---

### 3 기본 사용법
가장 기본적인 방법으로, 모듈 전체를 가져옵니다.
```python
import [사용할 파일명]
# 사용 예시: 파일명.함수이름()

import [사용할 파일명] as [약어]
# 사용 예시: 약어.함수이름()

from [사용할 파일명] import [함수이름]
# 사용 예시: 함수이름()

from [사용할 파일명] import *
